# FireCustomEvent

With Datalink, custom events can be fired to track certain triggers in your game. To implement such a system, `FireCustomEvent()` is used. 

<Code>
```lua
local ServerStorage = game:GetService("ServerStorage")

local DatalinkService = require(ServerStorage.DatalinkService)

DatalinkService:FireCustomEvent(
    eventCategory: string,
    {
        name = "CustomEvent1",
        data = "CustomEvent1Data"
    },
    {
        name = "CustomEvent2",
        data = "CustomEvent2Data"
    }
)

```
</Code>
