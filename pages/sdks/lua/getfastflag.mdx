# GetFastFlag

Datalink can manage feature rollouts through the Fast Flags feature. Fast Flags are percentage values indicating the amount of servers/users the feature should roll out to. In order to implement such a system, features can be controlled using `GetFastFlag()`

<Code>
```lua
local ServerStorage = game:GetService("ServerStorage")

local DatalinkService = require(ServerStorage.DatalinkService)

if DataLink:IsFeatureEnabled("SuperCoolRain") then
    StartRain()
end

```
</Code>
