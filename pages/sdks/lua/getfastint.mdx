# GetFastInt

Datalink can manage feature rollouts through the Fast Flags feature. Fast Flags are percentage values indicating the amount of servers/users the feature should roll out to.  

<Code>
```lua
local ServerStorage = game:GetService("ServerStorage")

local DatalinkService = require(ServerStorage.DatalinkService)

DatalinkService:GetFastInt(featureName: string, default: number)
-- default is the value to be returned in case the data fetching fails

```
</Code>
