# Yield until Authenticated

To wait for the Datalink SDK to be authenticated before accessing functionality in other scripts `YieldUntilDataLinkIsAuthenticated()` can be used. However, it is not mandatory as most Datalink functionalities already yield until the module is authenticated.

<Code>
```typescript
import { YieldUntilDataLinkIsAuthenticated } from '@datalinkhq/rbx-datalink'

YieldUntilDataLinkIsAuthenticated()

// Once authenticated, access other functions.

```
</Code>
