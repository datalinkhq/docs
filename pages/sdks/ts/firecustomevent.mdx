# FireCustomEvent

With Datalink, custom events can be fired to track certain triggers in your game. To implement such a system, `FireCustomEvent()` is used. 

<Code>
```typescript
import { FireCustomEvent } from '@datalinkhq/rbx-datalink'

FireCustomEvent(
    eventCategory: string, {
        name: "CustomEvent1",
        data: "CustomEvent1Data"
    }, {
        name: "CustomEvent2",
        data: "CustomEvent2Data"
    }
)

```
</Code>
