# FireLogEvent

Datalink can keep track of errors in your codebase similar to products such as [Sentry](https://sentry.io). This way, you can monitor and patch critical bugs and improve the mobility of your game.

<Code>
```lua
import { FireLogEvent } from '@datalinkhq/rbx-datalink'

try {
    InitializeGame();
} catch (err) {
    FireLogEvent(
        Enum.AnalyticsLogLevel[4],
        err, {
            details: "Game initialization failed!"
        }
    )
}

```
</Code>
