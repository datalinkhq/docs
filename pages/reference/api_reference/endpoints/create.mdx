# /create

Create an account.

### Request Body

<Code>
```json
{
    "id": 1, // Number
    "password": "CALLBACK_TOKEN", // String: GitHub OAuth callback token
}
```
</Code>

### Response
<Code>
```json
{
    "code": 200, // Number: IETF Status Code
    "status": "success", // String: Success
    "token": "ACCOUNT_TOKEN" // String: token
}
```
</Code>
